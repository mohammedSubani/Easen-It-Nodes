* Inputs : 

- Branches in level : number of branches at the end of each branch .
- Stopping level : a level of indentation at which the tree will stop creating
		   more branches .
- Max length : The maximum length for any branch since it is a random length branches tree .
- Root line : A curve (usually a line) which at its end the tree structure begins to branch .

* Outputs :

- List of lines : Their geometry is a tree like structure .

* Description : 

Create a random length branches recursive tree .
